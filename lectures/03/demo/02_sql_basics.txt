    "## NULL Handling and Error Cases\n",
    "\n",
    "NULL values are a common source of confusion in SQL. Let's see how to handle them and look at some common error cases:"
   ]
  },
  {
   "cell_type": "code",
   "execution_count": null,
   "metadata": {},
   "source": [
    "%%sql\n",
    "-- NULL handling in aggregates\n",
    "-- This query shows different ways to handle NULLs in aggregations\n",
    "SELECT \n",
    "    gender,\n",
    "    COUNT(*) as total_count,  -- Counts all rows, including NULLs\n",
    "    COUNT(education) as non_null_education,  -- Counts only non-NULL education values\n",
    "    AVG(age) as avg_age,  -- NULLs are excluded from average\n",
    "    AVG(CASE WHEN age IS NOT NULL THEN age ELSE 0 END) as avg_age_with_null_handling  -- Explicit NULL handling\n",
    "FROM demographics\n",
    "GROUP BY gender;"
   ]

    "## GROUP BY and Aggregates\n",
    "\n",
    "GROUP BY allows us to summarize data by groups. This is essential for data analysis:"
   ]
  },
  {
   "cell_type": "code",
   "execution_count": null,
   "metadata": {},
   "source": [
    "%%sql\n",
    "-- Count participants by gender\n",
    "-- Basic grouping and counting\n",
    "SELECT gender, COUNT(*) AS count\n",
    "FROM demographics\n",
    "GROUP BY gender;"
   ]
  },
  {
   "cell_type": "code",
   "execution_count": null,
   "metadata": {},
   "source": [
    "%%sql\n",
    "-- Calculate average age by race\n",
    "-- Shows how to use aggregate functions with GROUP BY\n",
    "SELECT race, AVG(age) AS avg_age\n",
    "FROM demographics\n",
    "GROUP BY race;"
   ]
  },
  {
   "cell_type": "code",
   "execution_count": null,
   "metadata": {},
   "source": [
    "%%sql\n",
    "-- Multiple aggregates\n",
    "-- Shows how to calculate multiple statistics in one query\n",
    "SELECT \n",
    "    race,\n",
    "    COUNT(*) AS count,  -- Number of participants\n",
    "    AVG(age) AS avg_age,  -- Average age\n",
    "    MIN(age) AS min_age,  -- Youngest participant\n",
    "    MAX(age) AS max_age  -- Oldest participant\n",
    "FROM demographics\n",
    "GROUP BY race;"
   ]
  },
  {
   "cell_type": "markdown",
   "metadata": {},
   "source": [
    "## HAVING Clause\n",
    "\n",
    "HAVING allows us to filter groups after aggregation. This is different from WHERE, which filters rows before grouping:"
   ]
  },
  {
   "cell_type": "code",
   "execution_count": null,
   "metadata": {},
   "source": [
    "%%sql\n",
    "-- Find races with more than 100 participants\n",
    "-- Shows how to filter groups based on aggregate values\n",
    "SELECT race, COUNT(*) AS count\n",
    "FROM demographics\n",
    "GROUP BY race\n",
    "HAVING COUNT(*) > 100;"
   ]
  },
  {
   "cell_type": "code",
   "execution_count": null,
   "metadata": {},
   "source": [
    "%%sql\n",
    "-- Find races with average age over 40\n",
    "-- Another example of filtering groups\n",
    "SELECT race, AVG(age) AS avg_age\n",
    "FROM demographics\n",
    "GROUP BY race\n",
    "HAVING AVG(age) > 40;"
   ]
