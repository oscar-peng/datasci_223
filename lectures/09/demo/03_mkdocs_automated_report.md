# Demo 3: Automated Report with MkDocs

This demo focuses on setting up an MkDocs site, embedding an Altair chart, and a Mermaid diagram.

**Task:**
1.  Initialize a new MkDocs project.
2.  Configure `mkdocs.yml` (e.g., with Material theme, Altair plugin).
3.  Create a Python script to generate an Altair chart (e.g., reusing from Demo 2) and save it as JSON in `docs/charts/`.
4.  Write a basic `docs/index.md` page.
5.  Embed the Altair chart using the `mkdocs-altair-plugin`.
6.  Embed a Mermaid diagram (e.g., a workflow for the data used in the report) directly into the Markdown.
7.  Build and serve the MkDocs site locally.
