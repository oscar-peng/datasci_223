# About This Report

This report is a demonstration project for **Lecture 09: Data Visualization, Diagramming, Reporting & Dashboards** in the Health Data Science Master's program.

## Purpose

The primary goal is to illustrate how to:
- Generate insightful data visualizations using Altair.
- Create clear process diagrams using Mermaid.
- Build an automated, shareable report using MkDocs.
- Integrate these components into a cohesive narrative.

## Authors

- Health Data Science Team (Students of the course)

## Tools Used

- Python
  - Pandas
  - NumPy
  - Altair
- MkDocs
  - Material for MkDocs theme
  - mkdocs-charts-plugin
- Mermaid (for diagrams)
